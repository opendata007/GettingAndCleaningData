
How does the run_analysis.R work


1. This script starts by loading the require libraries used in the script.

2. Reads the downloaded files

3. Creates a list of variables which are either mean or std deviavion mesurements

4. Subsets the dataset to included only relevant measurements

5. Merges the test and train datasets

6. Sets the variable names for the dataset to descriptive names

7. sets the activity names to descriptive names

8. calculates the mean for each variable by subject and activity

9. Writes the calculated data set to a text file
